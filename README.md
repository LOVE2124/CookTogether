# CookTogether

### Digital Community Cookbook Website

## 📌 Project Overview

**CookTogether** is a Digital Community Cookbook platform designed to help users explore, share, and save recipes contributed by the community.

The platform allows home cooks to submit their own recipes, discover dishes from around the world, and build a personal collection of their favorite meals.

This project was developed as part of the Innovative Assignment submission.

## 🎯 Project Objectives

- Provide a platform for community-driven recipe sharing
- Allow users to explore and filter recipes by category, time, and difficulty
- Enable users to save and manage their favorite recipes
- Create a professional, responsive web interface
- Demonstrate collaborative Git workflow using branches
